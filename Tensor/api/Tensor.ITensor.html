    Interface ITensor

    Type-neutral interface to Tensor<'T> of any type 'T.

    Namespace: Tensor
    Assembly: Tensor.dll
    Syntax
    interface ITensor
    Remarks

    These functions are useful for working with tensors of unknown types. For most use cases the functions provided by Tensor<'T> are better suited.
